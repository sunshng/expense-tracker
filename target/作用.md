# 记账APP - 项目文件夹作用说明

## 📂 项目总览

```
记账APP/
├── public/              # PWA 静态资源（图标等）
├── pwa-output/          # PWA 手机版构建输出
├── src/                 # Vue 3 前端源码（核心代码）
├── src-tauri/           # Tauri 桌面端（Rust 后端）
├── target/              # 安装包输出目录
├── CLAUDE.md            # 产品文档
├── package.json         # 项目配置
├── vite.config.js       # 构建工具配置
└── index.html           # 入口 HTML
```

---

## 📁 public/ — PWA 静态资源

存放 PWA（手机网页版）需要的图标文件，构建时直接复制到输出目录。

| 文件 | 说明 |
|------|------|
| `icons/icon-192x192.png` | PWA 小图标（手机主屏幕） |
| `icons/icon-512x512.png` | PWA 大图标（启动画面） |

---

## 📁 pwa-output/ — PWA 手机版构建输出

前端代码构建后的产物，可直接部署到任何网站服务器。当前已部署到 GitHub Pages。

| 文件/文件夹 | 说明 |
|-------------|------|
| `index.html` | 入口网页 |
| `manifest.webmanifest` | PWA 配置文件（应用名称、图标等） |
| `sw.js` | Service Worker（实现离线使用） |
| `registerSW.js` | 注册 Service Worker 的脚本 |
| `assets/` | 所有 JS/CSS 代码文件 |
| `icons/` | PWA 图标 |

---

## 📁 src/ — Vue 3 前端源码

这是 APP 的核心代码，所有界面和交互逻辑都在这里。同一份代码同时支持桌面端和手机端。

| 文件/文件夹 | 说明 |
|-------------|------|
| `main.js` | Vue 应用入口，加载路由、状态管理、UI 组件库 |
| `App.vue` | 根组件，包含桌面侧边栏 + 手机底部导航的响应式布局 |
| `assets/styles.css` | 全局样式，定义浅色/深色主题 |
| `router/index.js` | 路由配置，定义页面跳转规则 |
| `store/index.js` | Pinia 状态管理（主题、预算、账户、提醒） |
| `db/index.js` | 统一数据库接口，自动检测桌面端/PWA 环境 |
| `db/browser.js` | 浏览器端数据库（IndexedDB），PWA 使用 |
| `db/tauri.js` | 桌面端数据库（调用 Rust 后端），Tauri 使用 |
| `views/HomeView.vue` | 账单列表页（筛选、编辑、删除、导出） |
| `views/AddRecord.vue` | 记一笔页面（支出/收入、两级分类） |
| `views/Statistics.vue` | 统计分析页（饼图、趋势图、排行） |
| `views/SettingsView.vue` | 设置页（预算、账户、提醒、导入导出、主题） |

---

## 📁 src-tauri/ — Tauri 桌面端（Rust 后端）

桌面版专用的后端代码，用 Rust 语言编写，负责 SQLite 数据库操作和系统级功能。

| 文件/文件夹 | 说明 |
|-------------|------|
| `Cargo.toml` | Rust 项目配置（依赖库列表） |
| `tauri.conf.json` | Tauri 应用配置（窗口大小、打包设置） |
| `build.rs` | 编译脚本 |
| `src/main.rs` | Rust 程序入口 |
| `src/lib.rs` | 命令处理器（前端调用的后端函数） |
| `src/database.rs` | SQLite 数据库操作（建表、增删改查、统计） |
| `icons/` | 桌面应用图标 |

---

## 📁 target/ — 安装包输出目录

存放打包好的安装程序和文档。

| 文件 | 说明 |
|------|------|
| `记账APP_Setup.exe` | Windows 安装程序（3.6MB，一键安装） |
| `作用.md` | 本文档 |

---

## 📄 根目录文件

| 文件 | 说明 |
|------|------|
| `CLAUDE.md` | 产品文档（功能规划、分类体系、决策记录） |
| `package.json` | npm 配置（脚本命令、依赖包列表） |
| `vite.config.js` | Vite 构建配置（含 PWA 插件设置） |
| `index.html` | HTML 入口文件 |
| `.gitignore` | Git 忽略规则（排除 node_modules 等） |
| `installer.nsi` | NSIS 安装包脚本（生成 Setup.exe） |

---

## 🔄 数据流向

```
用户操作 (Vue 前端)
    ↓
src/db/index.js (统一接口)
    ↓
┌──────────────┬──────────────┐
│  PWA 环境     │  Tauri 环境   │
│  db/browser.js│  db/tauri.js │
│  IndexedDB    │  invoke()    │
│  (浏览器存储)  │      ↓       │
│               │  Rust 后端   │
│               │  SQLite 存储  │
└──────────────┴──────────────┘
```
